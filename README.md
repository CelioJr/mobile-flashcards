# Project Mobile FlashCards 

This project is focused on building flashcards using react native

## Platforms

| Platform | Tested | 
|:---------|:-------|

## Getting Started
To run this project locally, clone this repository and run the following commands:
 - `yarn`
 - `yarn start`
